# Microsoft Stream - links
Chromium-based extension that download a text file that contain every link's videos of a Microsoft Stream group.
  

### Installation on your browser:
```
- Download the directory.
- Activate developer mode on your browsers extensions setting.
- "Load decompressed" -> select this directory.
```
